# /ヘルプ - コマンド一覧

## 開発系
| コマンド | 説明 |
|----------|------|
| `/新機能` | 新しい機能を追加する |
| `/バグ修正` | 動かないものを直す |
| `/調整` | 見た目や細かい変更 |

## 確認・デプロイ系
| コマンド | 説明 |
|----------|------|
| `/確認` | ビルドできるかチェック |
| `/反映` | stagingにデプロイ（菅原さんだけ確認可能） |
| `/本番リリース` | mainにマージ（会員が使える本番反映） |

## 管理系
| コマンド | 説明 |
|----------|------|
| `/記録` | 今日の作業をPROJECT_CONTEXT.mdに記録 |
| `/状況` | 今のブランチや状態を確認 |
| `/ヘルプ` | このコマンド一覧を表示 |

## 普段の流れ

```
1. /新機能 or /バグ修正 or /調整 → 実装
2. /反映 → stagingで動作確認
3. /本番リリース → 会員が使える本番に反映
4. /記録 → 今日の作業を記録
```
