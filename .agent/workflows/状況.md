# /状況 ワークフロー

## 使い方
```
/状況
```

## 手順

1. **ブランチ確認**
   ```bash
   git branch --show-current
   ```

2. **直近のコミット確認**
   ```bash
   git log -3 --oneline
   ```

3. **未コミットの変更確認**
   ```bash
   git status
   ```

4. **PROJECT_CONTEXT.md 確認**
   - 最終更新日
   - 直近の作業内容

5. **報告**
   - 現在のブランチ
   - 直近の変更内容
   - 未コミットの変更があるか
   - 次に何をすべきかの提案
