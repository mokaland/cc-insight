## 補足情報（必要に応じて追加説明用）

### API呼び出しイメージ

```
POST https://cc-insight-app.vercel.app/api/invitations/create
Headers:
  Content-Type: application/json
  X-API-Key: {シークレットキー}
Body:
{
  "team": "{チームID}",  // fukugyou / taishoku / buppan
  "memo": "{メンバー名}"  // 管理用メモ（任意）
}
```

### レスポンス例

```json
{
  "success": true,
  "code": "ABC12345",

}
```
